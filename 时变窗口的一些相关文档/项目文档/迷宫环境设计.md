# 二维时变窗口迷宫环境设计

## 1. 任务定义

当前环境用于训练“连续路径上的无人机二维投影”穿越一类时变窗口迷宫，而不是只通关某一张固定图。

- 黑色墙体是静态障碍，任何接触都算失败。
- 白色区域是连续自由空间。
- 每个时变窗口由两端贴墙的线/折线/曲线构成，窗口线本体是黑色障碍。
- 每条窗口中间只有一个蓝色动态开口可通过；开口大小和位置随时间变化。
- 迷宫、窗口位置、窗口形态、开口时序都按 seed 程序生成。

该环境是开闭闸门、摆动隔断、窗帘缝隙等问题的二维投影。

## 2. 当前实现

```text
gap_step/window_maze_env.py
gap_step/preview_window_maze.py
gap_step/tests/test_window_maze_env.py
```

核心对象：

```text
WallRect         静态黑墙
ApertureWindow   墙到墙窗口，支持直线/折线/曲线形态
```

窗口合法性约束：

- 两端必须贴墙；
- 窗口线只布置在白色自由区；
- 窗口之间不能相交或冲突；
- 起点到终点存在拓扑可行路径。

## 3. 连续动作与碰撞

```python
action_space = Box(-1.0, 1.0, shape=(2,))
```

智能体是二维圆盘，当前正式实验参数：

```text
robot_radius = 0.18
max_step = 0.34
```

环境使用 swept-circle 检测整段连续路径。以下情况立即失败：

- 撞静态黑墙；
- 撞窗口黑线或关闭后的窗口实体；
- 出界；
- 动作后时间推进，当前位置被新相位窗口覆盖。

## 4. 课程与 C5

课程主线是：

```text
C1 -> C1_5 -> C2 -> C2A -> C2B -> C3 -> ... -> C4F -> C5
```

其中后半段桥接课程只用于让纯 PPO 更平稳地升阶。C5 仍保留：

- 全路径长迷宫；
- 6 个时变窗口；
- 直线、折线、曲线混合；
- 未见地图 seed；
- 开口时序不规则。

当前 C5 最窄开口下界为 `0.72`，这是为了把任务校准到“高难度但纯 PPO 可稳定学习”的区间。
