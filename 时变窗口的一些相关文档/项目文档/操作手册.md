# GAP-Step v4.6 实验操作手册

## 1. 环境准备

```bash
cd /home/jack/wyh/时变窗口
source /home/jack/anaconda3/etc/profile.d/conda.sh
conda activate wyh
```

检查测试：

```bash
pytest -q
```

## 2. 当前入口

所有入口都在 `gap_step/`：

```text
python -m gap_step.train
python -m gap_step.evaluate
python -m gap_step.visualize
```

当前主线是 GNN privileged teacher，不运行旧视觉学生、BC、启发式演示、`trainers/` 或 `scripts/`。

## 3. 观测约定

教师输入是 `GraphObs`，不是固定长度局部向量：

```text
global_features: [16]
node_features:   [num_nodes, 32]
node_type:       [num_nodes]
edge_index:      [2, num_edges]
edge_features:   [num_edges, 20]
```

图中包含 cell node、gate node、cell-cell topology edge、gate-cell edge 和 self-loop。gate 特征包含完整动力学、当前 safe、time-to-open 和 time-to-close。

## 4. 烟测训练

```bash
python -m gap_step.train --config gap_step/configs/train_teacher_smoke.yaml
```

预期输出：

```text
checkpoints/teacher_final.pt
checkpoints/teacher_best.pt
results/train_metrics.csv
```

smoke 只验证流程，不代表策略收敛。

## 5. 完整训练

```bash
python -m gap_step.train --config gap_step/configs/train_teacher_full.yaml
```

课程顺序：

```text
C1 -> C1_5 -> C2A -> C2B -> C3 -> C4 -> C5
```

adaptive promotion 仍使用 rollout success 和 deterministic train-validation 双条件。`teacher_best.pt` 根据 deterministic eval 更新。

## 6. 评估

默认评估 best checkpoint：

```bash
python -m gap_step.evaluate --checkpoint checkpoints/teacher_best.pt
```

按阶段评估：

```bash
python -m gap_step.evaluate --checkpoint checkpoints/teacher_best.pt --episodes 20 --stages C1,C1_5,C2A,C2B,C3,C4,C5
```

快速 smoke 评估：

```bash
python -m gap_step.evaluate --checkpoint checkpoints/teacher_best.pt --episodes 2 --stages C1,C1_5,C2A,C2B,C3,C4,C5 --output /tmp/gap_gnn_eval_smoke.csv
```

## 7. 结果解释

当前 GNN 重构的首要目标不是马上报告 C5 成功率，而是先验证：

1. C1/C1_5 是否稳定；
2. C2A 是否学会等待并过单动态 gate；
3. C2B 是否显著好于旧局部 teacher 的 deterministic 0%；
4. C2B 稳定后再推进 C3-C5。
